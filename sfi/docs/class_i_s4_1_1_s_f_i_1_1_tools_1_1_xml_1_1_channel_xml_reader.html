Parameters

capacityThe capacity of the channel, if it should be bounded.
Returns
The XML reader using the created channel.

The channel is created with the following settings:

  • ChannelOptions.AllowSynchronousContinuations true
  • ChannelOptions.SingleReader true
  • ChannelOptions.SingleWriter true
  • BoundedChannelOptions.FullMode (if capacity is provided) BoundedChannelFullMode.Wait
