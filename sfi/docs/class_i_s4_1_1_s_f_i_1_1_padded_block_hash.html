Detailed Description

Represents a chunked hash that stores a variable number of SHA-1 hashes to use to produce BitTorrent files. The hash has a specific structure, starting with 4 bytes storing the (int32) size of the padding (BitTorrentHashCache.FileInfo.Padding), followed by each block hash (BitTorrentHashCache.FileInfo.BlockHashes) and the hash of the last block, padded and unpadded (BitTorrentHashCache.FileInfo.LastHashPadded and BitTorrentHashCache.FileInfo.LastHash)
