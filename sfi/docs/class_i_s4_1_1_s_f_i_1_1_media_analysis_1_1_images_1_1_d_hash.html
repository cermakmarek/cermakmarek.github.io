Detailed Description

A dHash is a type of image hashing algorithm that encodes the difference between neighboring pixels in a scaled-down version of the input image. This version scales the image down to two 9×8 and 8×9 images and interlaces the resulting hash. The pixels are compared based on the result of Color.GetBrightness, and if the values are equally, the result differs between the two scaled-down variants.
