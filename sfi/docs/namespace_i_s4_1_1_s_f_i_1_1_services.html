◆ ContainerBehaviour

enum IS4.SFI.Services.ContainerBehaviour

Specifies the flags provided via AnalyzeInner

Enumerator
None 

No specific behaviour of the analysis.

FollowChildren 

Indicates that children nodes of the current node should be followed by the current analyzer.

BlockOther 

Indicates that multiple instances of this analyzer type should not enter this node, i.e. that the instance of IContainerAnalyzerProvider that was used to obtain this analyzer should not be used when matching this node as a root.

Specifies the formatting method used when converting the hash output to a URI.

Enumerator
Hex 

The bytes of the output are convered to uppercase hex characters.

Base32 

DataTools.Base32<TList>(TList, StringBuilder, string) is used to format the bytes.

Base58 

DataTools.Base58<TList>(TList, StringBuilder, string) is used to format the bytes.

Base64 

DataTools.Base64Url(ArraySegment<byte>, StringBuilder) is used to format the bytes.

Decimal 

The bytes are formatted as an unsigned decimal value.
